un programma java si salva in un file unicode, con il titolo dell'identificatore della dichiarazione della classe.

HelloWorld.java
```java
public class HelloWorld
{
	public static void main(String[] args)
	{
		System.out.print("Hello, World!");
		System.out.println();
	}
}
```
